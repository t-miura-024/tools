# ADR 形式

配置: `docs/adr/NNNN-slug.md`（連番）

## テンプレート

```md
---
status: accepted
---

# {短いタイトル}

{文脈、決めたこと、理由を 1〜3 文で書く。}
```

必要ならだけ次を足す:

- **Considered Options** — 捨てた案を覚えておく価値があるとき
- **Consequences** — 自明でない波及があるとき

## 作成条件

次の 3 つをすべて満たすときだけ ADR にする:

1. 後から覆すコストが高い
2. 文脈なしだと将来の読み手が驚く
3. 実在するトレードオフから選んだ

1 つでも欠けたら作らない。
